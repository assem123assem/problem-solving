```
let 5 liters container = A
let 3 liters container = B
```

## steps

```
1 - first we fill B till it is full
2 - then we pour all the water of B in A (container A has 2 liters left)
3 - we fill B till it's full again
4 - we pour the water of B in A again (A has 0 liters left, it is full, and B has 1 liters left )
5 - we empty A container
6 - pour the content of B (which is 1 litre now) in A
7 - fill container B again and pour it in A

```
